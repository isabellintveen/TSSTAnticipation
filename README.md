# TSSTAnticipation
Stress-related hypofrontality in depression might be affected by altered activation prior to the stress response
authors: Isabell Int-Veen, Ann-Christine Ehlis, Agnes Kroczek, Hendrik Laicher, Andreas Fallgatter and David Rosenbaum 

Here you find the SPSS code and the R scripts used for generating the plots. In addition, we provide the main script used for the fNIRS data analysis. However, the fNIRS preprocessing pipeline relies on a large number of interconnected MATLAB sub-scripts, which are not easily consolidated and shared in a usable format. For this reason, we are currently unable to provide the full preprocessing codebase.
